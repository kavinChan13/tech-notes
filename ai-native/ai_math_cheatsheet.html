01

线代必备

1.1 矩阵乘法 · 直觉

C[i,j] = Σ A[i,k] × B[k,j]

直觉：C 的 (i,j) 元素 = A 的第 i 行 · B 的第 j 列。Transformer 全部就是这个。

1.2 向量内积（点积）

a · b = Σ a_i × b_i = |a| × |b| × cos(θ)

直觉：a 和 b 相似度（cosine 把长度抹掉，只看方向相似度）。Attention 的 QKT 就是 query 和 key 的内积。

1.3 矩阵分解（SVD）

M = U × Σ × VT

直觉：任何矩阵能拆成"旋转 × 拉伸 × 旋转"。LoRA 用低秩分解（M ≈ B@A）的数学根基。

1.4 范数（Norm）

  • L1 norm：||x||_1 = Σ |x_i| — 绝对值和
  • L2 norm：||x||_2 = √(Σ x_i²) — 欧氏距离
  • Frobenius：矩阵的 L2 推广，||M||_F = √(Σ M_ij²)

1.5 矩阵求导（一些常用结果）

∂(Wx)/∂x = WT
∂(xTAx)/∂x = (A + AT)x

不需要会推，需要时直接查 Matrix Cookbook（PDF 免费）。

02

概率必备

2.1 期望（Expectation）

E[X] = Σ x · P(X=x)

就是加权平均。RL 里 expected reward 就是这。

2.2 方差（Variance）

Var(X) = E[(X - E[X])²]

方差 = 数据分散程度。RoPE 推导里方差 = d 是关键。

2.3 贝叶斯定理

P(A|B) = P(B|A) × P(A) / P(B)

直觉："看到证据 B 后 A 的概率"。所有"逆向推理"的根基。

2.4 KL Divergence · 两个分布有多远

KL(P || Q) = Σ P(x) × log(P(x) / Q(x))

直觉："用分布 Q 近似 P，损失多少信息"。DPO loss / VAE / RLHF 的 KL 约束都用。

关键性质：KL(P || Q) ≠ KL(Q || P)，不对称。前者 Q 不能漏 P 的高概率区，后者 P 不能漏 Q 的高概率区。

2.5 Softmax · 把分数变概率

softmax(x_i) = exp(x_i) / Σ exp(x_j)

直觉：所有 x_i 加大权重（exp），再归一化为 0-1。Attention 和 LLM 输出的标准做法。

2.6 交叉熵（Cross-Entropy）

H(P, Q) = -Σ P(x) × log Q(x)

直觉："用分布 Q 编码来自分布 P 的数据，平均需要多少 bit"。分类任务和 LLM next-token prediction 的标准 loss。

03

优化必备

3.1 SGD（Stochastic Gradient Descent）

θ ← θ - α × ∇L(θ)

直觉：参数 θ 朝 loss 下降最快的方向（负梯度）走一小步（学习率 α）。所有训练的基础。

3.2 Adam · 自适应学习率

SGD 升级版。维护两个 moving average：
① m = 梯度的 mean（"方向"）
② v = 梯度的 var（"幅度"）
然后 θ ← θ - α × m / (√v + ε)。2026 LLM 训练 99% 用 AdamW（Adam + weight decay 修正版）。

3.3 学习率 schedule

  • Warmup：开始时学习率从 0 慢慢上升（典型 1000 step）— 避免初期不稳
  • Cosine decay：之后按 cosine 曲线下降到 0 — Llama 训练标配

3.4 Gradient Clipping

梯度 norm 超过阈值（典型 1.0）就截断。防止梯度爆炸 / loss spike NaN 的关键。所有大模型训练必开。

3.5 Mixed Precision

FP16 / BF16 做 forward / backward（快 + 显存少），但保留 FP32 master weights 做 optimizer step（数值稳定）。2026 训练标配。

04

信息论必备

4.1 熵（Entropy）

H(X) = -Σ P(x) × log P(x)

直觉："分布的不确定性"。均匀分布熵最大，one-hot 熵 = 0。语言模型的 loss 接近 token 分布的熵。

4.2 Perplexity（困惑度）

PPL = exp(loss) = exp(cross-entropy)

直觉：模型预测下一个 token 时"有效选择数"。PPL = 10 意味着平均犹豫在 10 个候选之间。LLM 训练时核心 metric（越低越好）。

4.3 Mutual Information（互信息）

I(X; Y) = H(X) - H(X|Y)

"知道 Y 后 X 的不确定性减少多少"。评估 feature 跟 label 相关性用。
