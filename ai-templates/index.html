怎么用这些模板（推荐顺序）

  1. 启动新 AI feature 时：先写 AT01 PRD → 重点对齐 Eval / Failure mode / Cost / Risk 4 段
  2. 招新人时：开 role 前锁定 AT03 Scorecard → 每场面试官独立填分 → debrief
  3. 事故发生时：24 hr 内套 AT02 Postmortem 模板 → 5 维归因 → action items 跟到 done
  4. 季度规划时：用 AT04 OKR（4 类 pattern + 2 季度示例）· 配合 M07 第一季度章
  5. 启动 eval 体系时：用 AT05 Eval Plan（4 阶段 + 3 场景示例）· 配合 X12 详细方法论
  6. 谈预算时：用 AT06 Cost Model（3 cost 结构 + 1 完整 SaaS 案例）· 配合 M07 第一月决策 4
