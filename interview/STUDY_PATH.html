Q2🎯 标记的卡片跟普通卡片区别？
普通卡
1 个简短答案，覆盖基本要点
🎯 卡
3 档样例答案（mid 合格 / senior 加分 / staff Offer）
+ 失分模式（最常见的 3 个错误回答）
+ 追问树（面试官顺着哪些点会再问）
+ 真实案例。
打开任一卡片，看 "🎯 面试官视角 · 深度信号" 折叠区。
Q3怎么用模拟面试模式？
入口
任一卡片应用页面右上角点 "🎯 模拟面试" → 选维度（4 个 MVP 维度）→ 答题（开计时）→ 看 review + 自评 → 弱项列表自动累计
快捷键
Ctrl+Enter 提交 / Space 暂停 / Esc 退出
Q4URL 参数过滤是什么？
说明
本文档中的链接（如 cpp_interview_cards.html?cats=值类别）会自动在卡片应用里激活对应过滤器，你点开就只看到这些卡片。
支持参数
?ids=51,53,54 精确卡集 · ?cats=A,B 多分类 · ?diffs=high,mid 难度 · ?q=move 搜索
页面顶部会显示 banner 告知"已被外链筛选" + 一键清除
Q5内容更新频率？
主线
架构 / EM / C++ / PM — 季度大更
新热点

订阅
GitHub repo watch（每次大更新会发 release notes）
Q6我想给团队一起用怎么办？
直接用

fork 后改 README，加自己团队的 onboarding 路径；可以基于本 STUDY_PATH 改成"我们公司的 EM 路径 / 新人 90 天计划"
