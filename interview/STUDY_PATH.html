FAQ

Q1：我时间有限，最起码读哪几个文件？

按角色挑：

  • C++ Dev → 🎯 cpp 卡 60 张 + Modern C++ PPT
  • EM → 🎯 EM 卡 30 张 + Software Management Guide
  • PM → 🎯 PM 卡 20 张 + Dual-Track Overview

Q2：🎯 标记的卡片跟普通卡片区别？

普通卡：1 个简短答案。
🎯 卡：3 档样例答案（mid 合格 / senior 加分 / staff Offer）+ 失分模式 + 追问树 + 真实案例。打开任一卡片，看 "🎯 面试官视角 · 深度信号" 折叠区。

Q3：怎么用模拟面试模式？

任一卡片应用页面右上角点 "🎯 模拟面试" → 选维度（4 个 MVP 维度）→ 答题（开计时）→ 看 review + 自评 → 弱项列表自动累计。快捷键：Ctrl+Enter 提交 / Space 暂停 / Esc 退出。

Q4：URL 参数过滤是什么？

本文档中的链接（如 cpp_interview_cards.html?cats=值类别）会自动在卡片应用里激活对应过滤器，你点开就只看到这些卡片。支持 ?ids=51,53,54 精确卡集 / ?cats=A,B 多分类 / ?diffs=high,mid 难度 / ?q=move 搜索。页面顶部会显示 banner 告知"已被外链筛选" + 一键清除。

Q5：内容更新频率？

主线（架构 / EM / C++ / PM）季度大更，新热点（AI-Native / LLM tooling / RT-Linux）月度增量。订阅 GitHub repo watch。

Q6：我想给团队一起用怎么办？

整个 repo 是 static HTML，clone 后直接打开 index.html 即可。或者 fork 后改 README 加自己团队的 onboarding 路径。
